# Support

## Getting Help

### Bug Reports and Feature Requests

For bugs and feature requests, please use **[GitHub Issues](https://github.com/zivtech/joyus-ai/issues)**:

- Search existing issues before opening a new one
- Use the provided issue templates to include all relevant information
- Label your issue appropriately (bug, enhancement, documentation, etc.)

### Questions and General Discussion

For questions, ideas, and general discussion, please use **[GitHub Discussions](https://github.com/zivtech/joyus-ai/discussions)**:

- Ask questions about using or extending the platform
- Share ideas for new capabilities
- Connect with other contributors and users

### Contributing

If you want to contribute code, documentation, or other improvements, see [CONTRIBUTING.md](CONTRIBUTING.md) for guidelines on how to get started.

### Security Issues

Do **not** open public issues for security vulnerabilities. See [SECURITY.md](SECURITY.md) for the responsible disclosure process.

## Response Expectations

This is an open-source project maintained on a best-effort basis. We aim to respond to issues and discussions in a timely manner, but cannot guarantee specific response times for community support requests.
